

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test


# Run particular test case 
$ npm run test --filePath 
$ Ex - npm test src/user/user.service.spec.ts
```
